# MINE4201-202601-E07

Welcome to the **Group E07** organization for the course **MINE4201 – Sistemas de Recomendación** (2026-01) at Universidad de los Andes. This group built **Rapipeliculasbuenas**, a movie recommendation web app powered by the MovieLens dataset.

---

## 📦 Repositories

| Repository | Description | Language |
|---|---|---|
| [Taller-1-Notebooks](https://github.com/MINE4201-202601-E07/Taller-1-Notebooks) | Jupyter notebooks used for exploratory data analysis and model development on the MovieLens dataset | Jupyter Notebook |
| [Taller-1-Back](https://github.com/MINE4201-202601-E07/Taller-1-Back) | Python/Flask REST API backend that serves movie recommendations, handles user authentication, and stores data in SQLite | Python |
| [Taller-1-Front](https://github.com/MINE4201-202601-E07/Taller-1-Front) | TypeScript/React (Vite) frontend with a Netflix-like UI for browsing movies, rating them, and receiving personalized recommendations | TypeScript |
